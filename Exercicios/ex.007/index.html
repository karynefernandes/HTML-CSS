Negrito/Destaque

Itálico/Ênfase

Nesta frase temos um termo em italico usando a tag "i" (não semântica)

Nesta frase, temos um em termo em Ênfase usando a tag "EM" (semântica)

Texto marcado

Podemos usar a tag "MARK" para criar um texto marcado.

Aqui temos outra forma de mudar a cor da marcação de texto.

Texto grande e pequeno

Podemos usar a tag "BIG" (obsoleta) para textos grandes e a tag "SMALL" para textos pequenos.

Texto deletado

Texto excluido, é para indicar que o mesmo deve ser lido mas não considerado.

Texto inserido

Usado para dar Ênfase e indicar que foi adicionado depois. Podemos usar tambem a tag "U" para sublinhado (não semântica)
